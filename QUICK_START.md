# Quick Start Guide

This guide will help you get your GEEKRIA Mini Macro Keyboard up and running in minutes.

## Step 1: Unbox and Connect

1. Remove the keyboard, USB cable, USB flash drive, and anti-slip pads from the package
2. Attach the 4 anti-slip pads to the bottom of the keyboard
3. Connect the USB cable to the keyboard
4. Plug the other end into your computer's USB port
5. The keyboard LED should light up - you're connected!

## Step 2: Test Default Functions

The keyboard comes pre-programmed with useful defaults:

- **4 Keys:** Act as arrow keys (↑ ↓ ← →)
- **Knob Turn:** Controls system volume
- **Knob Press:** Mutes/unmutes audio

Try these out to ensure everything works!

## Step 3: Install Configuration Software (Optional)

If you want to customize the keys:

### For Windows:
1. Extract the RAR file from this repository or the USB flash drive
2. Run `MINI_KEYBOARD.exe`
3. If Windows blocks it, click "More info" → "Run anyway"

### For MacOS:
1. Extract the ZIP file from this repository or the USB flash drive
2. Install `MINI_KEYBOARD.pkg`
3. If blocked, go to System Preferences → Security & Privacy → "Open Anyway"

## Step 4: Customize Your Keys

1. Launch the MINI_KEYBOARD application
2. Click on any key in the interface
3. Assign your desired function:
   - Keyboard shortcuts (Ctrl+C, Alt+Tab, etc.)
   - Media controls
   - Application shortcuts
   - Text macros
4. Click "Save" to apply

## Common First-Time Setups

### For Gaming
- Key 1: Ability/Item slot
- Key 2: Push-to-talk
- Key 3: Quick emote
- Key 4: Screenshot
- Knob: Game volume

### For Productivity
- Key 1: Copy (Ctrl+C)
- Key 2: Paste (Ctrl+V)
- Key 3: Undo (Ctrl+Z)
- Key 4: Save (Ctrl+S)
- Knob: System volume

### For Video Editing
- Key 1: Play/Pause
- Key 2: Cut
- Key 3: Undo
- Key 4: Render
- Knob: Timeline zoom
