## üîß Troubleshooting

### Keyboard Not Detected

- Try a different USB port
- Restart your computer
- Check if the LED lights up when connected
- Test the USB cable with another device

### Software Won't Open

**Windows:**
- Run as Administrator (right-click ‚Üí "Run as administrator")
- Add exception to antivirus
- Install Visual C++ Redistributables if prompted

**MacOS:**
- Check Security & Privacy settings
- Ensure you have administrator privileges
- Try reinstalling the package

### Keys Not Working After Programming

- Verify the configuration in the software
- Try resetting to default settings
- Disconnect and reconnect the keyboard
- Restart the configuration software

### Antivirus Blocking Software

- This is a false positive common with low-level hardware control software
- Add the software folder to your antivirus exceptions
- The software is safe - it only communicates with the keyboard hardware
